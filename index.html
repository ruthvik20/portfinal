Project

Farmers Connect project

project details

Intro

Farmers connect is cross platform mobile application built using react native as front end and firebase as back end. This project focuses on solving many farmer issues that they are facing in the current situations like knowing crop information, pesticides usage, fertilizers usage and etc. The team of 6 members are involved to create this app. This app is developed by following scrum methodology. 8 Sprints, each sprint is of week are planned to complete this app including training.

Technologies that we are learning

Frontend

React native

We have used react native tools and functionalities by using the environment of Expo, which is easy and fast way of implementation of app.

Backend

Firebase

We have used Firebase for storing and retrieving of data whenever necessary for the app funtions.

Code versioning system

Github

We have used the github for pushing and branching of code to the main branch.
